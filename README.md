# 校聚活动平台

## 项目介绍

校园活动平台，含活动发布、参与、搜索、分类、管理、评价、留言。PC端与移动端双端。

原项目开发代码提交地址（即本人gitee项目地址） https://gitee.com/h0812/activity-platform.git

## 技术栈

* PC前端HTML+CSS+JavaScript，无框架。

* 移动端前端HTML+CSS+Vue框架。

* Java后端框架Springboot+MyBatis-Plus
